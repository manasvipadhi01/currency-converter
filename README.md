# Currency Converter

This is a simple currency converter built using HTML, CSS and JavaScript.

## Features
- Convert between multiple currencies
- Live exchange rates using API
- Swap currencies
- Dark mode
- Conversion history
- Searchable currency list
- Mobile responsive design
- Exchange rate chart
- Easy use
  
## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js
- Exchange Rate API

## How to Run
1. Download the project
2. Open index.html in a browser
